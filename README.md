# 🌞 Solar Power Forecasting Using Machine Learning Techniques

This project focuses on predicting solar power generation using machine learning and time-series forecasting techniques.

## 📌 Features
- Linear Regression
- Bayesian Regression
- XGBoost
- ARIMA
- SARIMA
- SARIMAX
- Streamlit-based prediction dashboard

---

## 📊 Exploratory Data Analysis
The project includes:
- Trend analysis
- Seasonality analysis
- ACF/PACF plots
- Correlation analysis
- Sensitivity analysis
- Ground truth vs prediction comparison

All detailed experiments and visualizations are available in the Jupyter Notebook.

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Statsmodels
- TensorFlow/Keras
- Streamlit
- Matplotlib

## 🚀 Run Locally

### Clone repository

```bash
git clone <repo-link>
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run Streamlit app

```bash
streamlit run app.py
```
